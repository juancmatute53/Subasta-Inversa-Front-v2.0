HOLAMUNDO!

¡Hola! Quiero contarte sobre un proyecto muy chévere que hemos creado los estudiantes egresados del Instituto Superior Tecnológico del Azuay. Resulta que hemos desarrollado un sistema que permite hacer subastas inversas, lo cual básicamente significa que diferentes proveedores compiten para ofrecer el precio más bajo posible para prestar un servicio a una entidad pública. Estoy muy contento porque este sistema es genial ya que la entidad pública puede obtener un precio mucho más competitivo para el servicio que necesita. Además, este proyecto es el resultado de un examen complexivo que teníamos que pasar para poner en práctica todos nuestros conocimientos en tecnología y desarrollo de software. ¡Esperemos que te guste!
